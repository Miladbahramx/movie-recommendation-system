🎬 Movie Recommendation System

A movie recommendation system built using NLP and Machine Learning.

Features

- Movie recommendation based on content similarity
- Interactive web application with Streamlit
- Search movies and get top 5 recommendations

Tech Stack

- Python
- Pandas
- Scikit-Learn
- NLP
- Streamlit
